# 个人博客网站

这是一个使用 Next.js 和 Contentful CMS 构建的个人博客网站。

## 功能特性

- 响应式设计，适配各种设备
- 博客文章列表展示
- 单篇文章详情页面
- 关于页面
- 使用 Contentful CMS 管理内容
- 使用 Tailwind CSS 进行样式设计
